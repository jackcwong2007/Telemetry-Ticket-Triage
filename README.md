# Telemetry Ticket Triage (TTT) [demo](https://youtu.be/mszYz0ilzEg)

Telemetry & Ticket Triage (TTT) is a full-stack automated data pipeline that ingests, cleans, and routes messy customer support tickets in real-time. Built with a React frontend and a FastAPI backend, the system uses Cohere’s language models to securely strip out private personal data (PII) and instantly classify issues into structured priority queues based on urgency and technical domain.

Frontend: React(JS)/Vite, Backend: Python with FastAPI, Model: Cohere ClientV2 SDK (command-r-plus-08-2024)
